Quick Start

  1. Ensure Weaviate is running on port 27501 (auto-provisioned on first vector operation)
  2. Configure embedding provider in appsettings.json (default: Ollama with all-minilm)
  3. Use MCP endpoints from Claude Desktop or Cline
  4. Access REST API at http://localhost:27500/api/projects
