# Sylin.Koan.App

Reference this bundle for a Koan web application. It composes the independently versioned
`Sylin.Koan` foundation and `Sylin.Koan.Web` at versions verified together by the release clean room.

```xml
<PackageReference Include="Sylin.Koan.App" Version="0.17.*" />
```

Application code remains the standard four-line `AddKoan()` bootstrap; add provider packages to select
the application's infrastructure.
