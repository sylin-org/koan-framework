# Koan Framework Translation Service

Welcome to the **PolyglotShop** translation service powered by the Koan Framework.

## Features

- AI-powered translation using Ollama
- Support for multiple languages
- Drag-and-drop file translation
- Real-time translation with visual feedback

## How to Use

1. Select source and target languages
2. Enter text or **drag and drop a .txt or .md file**
3. Click Translate or press Ctrl+Enter
4. Copy the translated text

Built with Koan Services architecture and Entity-First patterns.
