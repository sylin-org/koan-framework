GUID v7 Auto-Generation

The Koan Framework automatically generates GUID v7 identifiers for Entity<T> instances. GUID v7 provides several advantages over traditional GUIDs:

Benefits of GUID v7
  • Sortable: Maintains creation order
  • Distributed: No coordination required
  • Collision-resistant: Cryptographically secure
  • Database-friendly: Excellent clustering properties
Framework Implementation
  • Automatic: Generated on first access
  • Lazy: Only when needed
  • Provider-agnostic: Works with all storage backends
  • Deterministic: Same ID across provider switches
